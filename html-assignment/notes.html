HTML Tags
Tags Description
<h1> Heading tag- for the most important heading of the page.
<em> Emphasis tag - A string of emphasized, slightly more important text.
<strong> Strong tag- A string of highly emphasized, much more important text.
<p> Paragraph tag- Used to enclose and define paragraphs of text.
<abbr> Abbreviation tag- An acronym or abbreviation, attribute contains the expanded version.
